# CRS-12 E-Portfolio Website

## Overview
CRS-12 is a responsive cybersecurity portfolio website developed to showcase projects, certifications, technical skills, and professional achievements.

## Objectives
- Build a personal cybersecurity portfolio.
- Present projects in a professional format.
- Provide easy access to GitHub and LinkedIn profiles.
- Implement modern UI/UX design principles.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- GitHub Pages

## Features
- Responsive Design
- Smooth Scrolling Navigation
- Modern Glassmorphism UI
- Interactive Project Cards
- GitHub Integration
- LinkedIn Integration
- Mobile-Friendly Layout

## Sections
- Home
- About
- Skills
- Projects
- Certifications
- Contact

## Skills Demonstrated
- Front-End Development
- Responsive Web Design
- UI/UX Design
- GitHub Deployment
- Personal Branding

## Deployment
The website is deployed using GitHub Pages.

## Results
Successfully developed and deployed a professional cybersecurity portfolio website showcasing projects and technical expertise.
